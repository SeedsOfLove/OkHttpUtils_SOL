# OkHttpUtils_SOL
网络请求框架类

## 使用方式
### Step 1. Add the JitPack repository to your build file
```
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
```
### Step 2. Add the dependency
```
dependencies {
        implementation 'com.github.SeedsOfLove:OkHttpUtils_SOL:1.0.0'
	}
```
